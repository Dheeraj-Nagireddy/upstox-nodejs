Upstox

Upstox

API client class. You need to initialise a single instance of this class per api_key. This module provides an easy to use wrapper over the HTTP APIs. The HTTP calls have been converted to methods and their JSON responses.


var Upstox = require("Upstox");
var up = new Upstox("your_apikey", "your_apiSecretKey");
up.getSessionToken("clientId", "password")
   .then(function(response) {
        init();
    })
   .catch(function(err) {
        console.log(err.response);
    })
